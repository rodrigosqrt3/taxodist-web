taxodist

Taxonomic Distance and Phylogenetic Lineage Computation

taxodist

An R and Python package for computing phylogenetic distances between any two taxa. Given two taxon names, taxodist retrieves their full hierarchical lineages from The Taxonomicon and computes a dissimilarity index based on the depth of their most recent common ancestor.

Get started Reference GitHub

Distance metric

The distance between taxa A and B is the reciprocal of the depth of their most recent common ancestor (MRCA) in The Taxonomicon hierarchy. The deeper the shared ancestor, the smaller the distance. Taxa diverging at the same node are always equidistant from any third taxon, regardless of lineage depth below the split.

d(A,B) = 1 / depth(MRCA)

Overview

Core functionality

Lineage resolution

Lineage data from The Taxonomicon (Systema Naturae 2000), with substantially deeper hierarchical resolution than most programmatic sources — up to 70 nodes for well-studied clades.

Distance computation

Pairwise distances between any two taxa, or a full symmetric distance matrix for a set. Returns standard dist objects compatible with the R and Python ecosystems.

Multivariate analysis

Wrapper functions for hierarchical clustering, PCoA ordination, and heatmap visualisation. Compatible with vegan and ape in R, and scipy in Python.

Lineage utilities

Retrieve full lineages, identify MRCAs, trace taxonomic paths, test clade membership, and filter lists of taxa by clade.

Session caching

Lineages are cached in memory after first retrieval. The cache can be serialised to disk and restored in future sessions, avoiding repeated network requests on large datasets.

R and Python

The API is equivalent across both languages, supporting consistent workflows in multi-language environments. Available on CRAN and PyPI.

Interactive web application

A Shiny web application providing access to the main package functions through an interactive interface — pairwise distances, distance matrices, lineage explorer, closest-relative ranking, coverage checking, and clade filtering.

Installation and basic usage

  • R
  • Python
install.packages("taxodist")
library(taxodist)

# Pairwise distance
taxo_distance("Tyrannosaurus", "Velociraptor")
#> Distance : 0.01538   MRCA: Tyrannoraptora (depth 65)

taxo_distance("Tyrannosaurus", "Homo")
#> Distance : 0.02778   MRCA: Amniota (depth 36)

taxo_distance("Tyrannosaurus", "Quercus")
#> Distance : 0.25000   MRCA: discaria (depth 4)

# Distance matrix and clustering
taxa <- c("Tyrannosaurus", "Velociraptor", "Triceratops",
          "Homo sapiens", "Panthera leo", "Quercus robur")
mat <- distance_matrix(taxa)
cl  <- taxo_cluster(taxa)
plot(cl)
pip install taxodist
import taxodist as td

# Pairwise distance
td.taxo_distance("Tyrannosaurus", "Velociraptor")
# distance: 0.01538   mrca: Tyrannoraptora (depth 65)

td.taxo_distance("Tyrannosaurus", "Homo")
# distance: 0.02778   mrca: Amniota (depth 36)

# Distance matrix and clustering
taxa = ["Tyrannosaurus", "Velociraptor", "Triceratops",
        "Homo sapiens", "Panthera leo", "Quercus robur"]
mat = td.distance_matrix(taxa)
cl  = td.taxo_cluster(taxa)
cl.plot()

Functions

API reference

Distance computation

  • taxo_distance()Pairwise distance between two taxa
  • distance_matrix()Full symmetric distance matrix for a set of taxa
  • focal_distances()Distances from one focal taxon to a community
  • closest_relative()Nearest taxon among a set of candidates
  • mrca()Most recent common ancestor of two taxa

Lineage and exploration

  • get_lineage()Full lineage by taxon name
  • get_lineage_by_id()Full lineage by Taxonomicon numeric ID
  • get_taxonomicon_id()Resolve taxon name to numeric ID
  • taxo_search()Search Taxonomicon; handles homonyms
  • compare_lineages()Side-by-side lineage comparison at the MRCA
  • shared_clades()Shared lineage trunk between two taxa
  • taxo_path()Node-by-node path from taxon A to taxon B
  • lineage_depth()Number of nodes in a taxon's lineage
  • is_member()Test whether a taxon belongs to a clade
  • filter_clade()Retain only taxa belonging to a given clade
  • check_coverage()Verify which taxa are present in Taxonomicon

Multivariate analysis

  • taxo_cluster()Hierarchical clustering (UPGMA)
  • taxo_ordinate()Principal coordinate analysis (PCoA)
  • taxo_heatmap()Distance heatmap with marginal dendrograms

Cache management

  • clear_cache()Remove all entries from the session cache
  • save_cache()Serialise the cache to an .rds file
  • load_cache()Restore a previously saved cache from disk
  • cache_info()Report cache contents and memory use

Data source and citation

All lineage data is sourced from The Taxonomicon, based on Systema Naturae 2000 (Brands, S.J., 1989 onwards). Please cite this resource in any published work that uses taxodist.

Brands, S.J. (1989 onwards). Systema Naturae 2000. Amsterdam, The Netherlands. Retrieved from The Taxonomicon, http://taxonomicon.taxonomy.nl.
